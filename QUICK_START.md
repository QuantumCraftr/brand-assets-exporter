# Quick Start

1. Placez vos fichiers SVG dans le dossier `exports/`.
2. Installez les dépendances :
   ```
   npm install
   ```
3. Lancez le script :
   ```
   node export-assets.js
   ```
4. Récupérez vos assets dans le dossier `output/`.

Pour toute personnalisation, modifiez la configuration dans `export-assets.js`.
